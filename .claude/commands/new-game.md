Thumby Color の新規ゲームプロジェクトを作成する。

ゲーム名: $ARGUMENTS

以下の4ファイルを作成すること。

## $ARGUMENTS/main.py

CLAUDE.md の「基本パターン」→「クラス継承型」のコードをそのままボイラープレートとして使うこと。

## $ARGUMENTS/CLAUDE.md

CLAUDE.md の「各プロジェクトのCLAUDE.md に書くべき内容」のテンプレートを使い、
`[ゲーム名]` の箇所を `$ARGUMENTS` に置き換えて作成すること。

## $ARGUMENTS/.claude/settings.json

MicroPython コードレビュー用の Stop hook:

```json
{
  "hooks": {
    "Stop": [
      {
        "hooks": [
          {
            "type": "agent",
            "prompt": "このセッションで .py ファイルが編集されていれば、../CLAUDE.md の「コーディング規約」セクションを参照してレビューしてください。編集がなければ何もしないでください。問題がなければ「レビューOK」とだけ報告してください。問題があれば該当箇所と修正案を具体的に示してください。",
            "timeout": 120
          }
        ]
      }
    ]
  }
}
```

## $ARGUMENTS/.claude/commands/review.md

手動でレビューを実行するためのコマンド:

```
Agent ツールを使ってサブエージェントを起動し、以下の指示でレビューを実行してください。

---
このプロジェクトの .py ファイルをすべて読み、../CLAUDE.md の「コーディング規約」セクションを参照してレビューしてください。

問題がなければ「レビューOK」とだけ報告してください。問題があれば該当ファイル・該当箇所と修正案を具体的に示してください。
---
```

作成完了後、生成したファイルの一覧を表示すること。
